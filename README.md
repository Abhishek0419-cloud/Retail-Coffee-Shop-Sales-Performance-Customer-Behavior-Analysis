# Retail Coffee Shop Sales Performance Analysis

## Project Overview
This project analyzes retail coffee shop sales data to identify revenue trends, customer purchasing behavior, and product performance insights to support business decision-making.

## Tools & Techniques Used
• Microsoft Excel  
• Pivot Tables & Pivot Charts  
• XLOOKUP & IF / SWITCH functions  
• Data Cleaning & Transformation  
• Interactive Dashboard with Slicers  

## Key Analysis Performed
• Monthly & yearly sales trend analysis  
• Country-wise revenue comparison  
• Product performance by roast type and size  
• Top 5 customer contribution analysis  
• Loyalty card impact on sales  

## Business Impact
The dashboard enables stakeholders to monitor KPIs dynamically, identify high-performing products and customers, and make data-driven decisions regarding inventory planning and targeted promotions.
